## Prerequisites

- [Claude Desktop](https://claude.ai/download) installed (macOS or Windows)
- A sheets-mcp account at [sheets-mcp-xi.vercel.app](https://sheets-mcp-xi.vercel.app)

---

### Step 1 — Open the Claude Desktop config file
**macOS:**
    ```bash
    open ~/Library/Application\ Support/Claude/
    ```

    **Windows:**
    ```
    %APPDATA%\Claude\
    ```

    Open `claude_desktop_config.json` in your editor. If it does not exist, create it.

### Step 2 — Add the sheets-mcp server
```json
    {
      "mcpServers": {
        "sheets-mcp": {
          "url": "https://sheets-mcp-xi.vercel.app/api/mcp"
        }
      }
    }
    ```

    If you already have other MCP servers, add `sheets-mcp` as a new key inside `mcpServers`.

### Step 3 — Restart Claude Desktop
Fully quit and relaunch Claude Desktop. On restart, Claude detects the new MCP server and initiates the OIDC auth handshake — it discovers the authorization endpoint from `/.well-known/oauth-authorization-server` automatically.

### Step 4 — Complete Google OAuth consent
Claude opens a browser window to `sheets-mcp-xi.vercel.app/authorize`. The consent screen shows:
    - The requesting client name ("Claude" or "An AI Assistant")
    - Your currently signed-in Google account email
    - The exact permissions being granted (Sheets read/write, Drive read)

    Click **Allow**. The browser redirects back and Claude closes the window automatically.

    > **Note**
>
> You can switch Google accounts from this screen by clicking **Switch account?** — sheets-mcp will sign you out and redirect back through the login flow with the new account.

### Step 5 — Verify the connection
In Claude, type:

    > "List all my Google spreadsheets"

    Claude calls `list_spreadsheets` and returns your Drive files. You are connected.

---

## Switching Google accounts

1. Go to [sheets-mcp-xi.vercel.app/dashboard](https://sheets-mcp-xi.vercel.app/dashboard)
2. Click **Sign out**
3. Restart Claude Desktop
4. Complete the OAuth consent flow with the new account

sheets-mcp uses `prompt: "select_account consent"` on the Google OAuth screen — the account picker is always shown.

## Troubleshooting

| Symptom | Fix |
|---|---|
| `list_spreadsheets` returns `[]` | Auth handshake not completed. Restart Claude and re-authorize. |
| Tools not appearing in Claude | Check that `claude_desktop_config.json` is valid JSON — use [jsonlint.com](https://jsonlint.com). |
| Google consent screen not opening | Claude opens the system default browser. Check your default browser setting. |
| `needsAuth: true` in tool response | Refresh token was revoked. Sign out from the dashboard and re-authorize. |
| Dashboard shows "Connect Google Account" banner | You signed in with email+password but haven't linked Google yet. Click the banner. |
